# Countdown Timer - Legal Documents

This directory contains the legal documentation for the Countdown Timer mobile application.

## Files

- **index.html** - Landing page with links to legal documents
- **privacy-policy.html** - Privacy Policy
- **terms-and-conditions.html** - Terms and Conditions

## GitHub Pages Setup

These files are hosted on GitHub Pages and can be accessed at:

## Local Testing

To test these pages locally, simply open the `index.html` file in a web browser.
